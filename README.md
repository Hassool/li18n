## ğŸŒ RTL Support

RTL languages (like Arabic or Hebrew) are detected automatically.

The system updates the `<html>` tag with:
- `dir="rtl"`
- `.rtl` class for CSS styling

**Example CSS:**

```css
.container {
  text-align: left;
}

.rtl .container {
  text-align: right;
  direction: rtl;
}
```
